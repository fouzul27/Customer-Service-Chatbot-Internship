# Customer Service Chatbot Internship Project

## Overview

This repository contains six internship tasks implemented as extensions of a Customer Service Chatbot project as part of the ElevanceSkills Internship Program.

The project demonstrates advanced Generative AI capabilities including:

* Dynamic Knowledge Base Expansion
* Multimodal AI Assistant
* Medical Question Answering
* Domain Expert Research Assistant
* Sentiment-Aware Customer Support
* Multilingual Conversations

All tasks are implemented as enhancements to the Customer Service Chatbot architecture using Retrieval-Augmented Generation (RAG), Vector Databases, Large Language Models, and Natural Language Processing techniques.
